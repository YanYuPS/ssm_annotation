# ssm_annotation
ssm

1. 表
登录表：login
角色表：role

管理员表：admin
学生表：student
老师表：teacher

老师教课表：teacher_course
学生选课表：student_course

***

2. 使用
学生登录--只查看自己的成绩
老师登录--查看和修改自己学生的成绩
管理员登录--查看和修改所有学生的成绩，删除和增加学生选课信息

登录地址：http://localhost:8080/ssm_annotation_web/

使用前：webapp下安装vue、axios
--npm init -y
--npm install vue --save
--npm install axios
